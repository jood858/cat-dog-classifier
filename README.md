This is a simple image classifier using Google's Teachable Machine and TensorFlow.js.

1. Click "Start"
2. Upload an image
3. See the result!

## Files
- `index.html`: Main page
- `model/`: Trained model files exported from Teachable Machine

## Tools Used
- Teachable Machine
- TensorFlow.js
